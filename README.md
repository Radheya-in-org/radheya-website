# Radheya (రాధేయ)

**Solving Tomorrow, Today.**

Official website for Radheya — a purpose-driven software company that builds products solving real-world problems and provides custom software development services.

Named after Karna from the Mahabharata — the warrior of humble origins who rose to greatness through purpose, skill, and unwavering conviction.

🌐 [radheya.in](https://radheya.in)

---

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Fonts:** Space Grotesk, DM Sans, Outfit (Google Fonts)
- **Deployment:** Vercel

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
